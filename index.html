App Features

This application will focus on script, file, and url viruses. Additionally, this detection application will continue to be developed and updated. Therefore, always follow us.

SQL injection

This app is equipped to detect SQL Injection attempts, which involve manipulating or injecting malicious SQL code into database queries.

Cross Site Scripting

This app employs security measures to detect Cross Site Scripting (XSS) attacks, which involve injecting malicious scripts into web pages viewed by other users.

Remote Code Execution

This app incorporates safeguards to detect RCE attacks, which involve executing malicious code on a target server from a remote location
